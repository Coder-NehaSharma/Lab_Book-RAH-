## ðŸ“š Curriculum Overview

The manual covers a complete beginner-to-intermediate robotics journey:
1.  **Introduction:** What is a robot?
2.  **Electronics:** Components (Resistors, Capacitors), Ohm's Law (Calculator included).
3.  **Arduino UNO:** Board layout, pins, and first sketch.
4.  **Sensors:** Ultrasonic, IR, PIR, and more.
5.  **Actuators:** Motors, Servos, and Drivers (L293D/L298N).
6.  **Coding Basics:** Variables, Loops, Functions, and Logic.
7.  **Projects:** Line Follower, Obstacle Avoider, and Innovation Challenges.
